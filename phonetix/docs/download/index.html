Unpacking

On Unix download the ".tar.bz2" or ".tgz" archive. These are bzipped (or gzipped) tar archives, which can be handled the usual way using tar and the bzip2 or GNU gzip programs. Note that the ".tar.bz2" archive provides the best compression and fastest download times.

On Windows use the ".zip" archive. It may be unpacked with the WinZip tool. Many other tools for handling ZIP archives are available and should also work. WinZip can also handle the ".tgz" tar archive. Note that the ".tar.bz2" file is smaller than the other archives. On Windows you may want to install the bzip2 tools as well.
