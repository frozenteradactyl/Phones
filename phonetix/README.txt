
                                Phonetix


  What is it?
  -----------

  Phonetix is a small, but useful class-library of phonetic algorithms
  written entirely in Java.
  It includes implementations of Soundex, Metaphone, and DoubleMetaphone.


  The Latest Version
  ------------------

  Details of the latest version can be found on tangentum technologies'
  website "http://www.tangentum.com".


  Documentation
  -------------

  Documentation is available in HTML format, and can be found in the
  "docs/" directory.


  Software
  --------

  The class-files are contained in the ".jar" file in the "lib/"
  directory. The file "phonetix-0.2.jar" contains the phonetic
  algorithms, and adapters to the Lucene search-engine.
  The sources can be found in the file "src.zip" in this directory.
  To rebuild the class-files, unpack "src.zip" and run "ant jars" in
  directory "src/". The "ant"-tool can be found on the website of
  Apache's Jakarta Project at "http://jakarta.apache.org".
